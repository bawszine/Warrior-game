# Warrior Guard - 2D Platformer Defense Game

## 🎮 About The Game
Warrior Guard is a simple 2D platformer defense game built with Unity and C#.

The main objective of the game is to protect the object located in the center of the map from incoming enemies. The player controls a warrior character who can move, jump, and attack enemies.

The goal of the game is to survive as long as possible and achieve the highest number of enemy kills while protecting the object in the center.

If the player dies or the object is destroyed, the game ends and the player can restart the level.

---

## 🕹️ Controls
- **A / D** → Move Left / Right  
- **Space** → Jump  
- **Left Mouse Click** → Attack  

---

## 🎯 Objective
- Protect the object in the center
- Defeat enemies
- Survive as long as possible
- Get the highest kill count
